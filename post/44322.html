Atom插件

Feb 18 2018

atom-ternjs
atom-beatify　　　　html/css/js等都能格式化，必装
prettier-atom　　主要js美化，还顺带一些纠错
open-in-browser　　浏览器打开
emmet　　这个不多说了
file-icons　　侧边栏图标
highlight-selectd　　高亮行
Sublime Block Comment　　块注释
language-babel 支持 JavaScript ES201x，React JSX，Flow和GraphQL语法。
atom-react-autocomplete　　项目内，组件名及状态的自动补全
language-javascript-jsx　　JavaScript, ES6, ES7, React JSX, Flow支持，还会高亮jsx
react-es6-snippets　　es6写法的react snippet，挺实用
react-snippets　　和上面的搭配“口味更加”
js-func-viewer　　查看函数的结构
atom-react-native-autocomplete　　RN的智能补全，react开发也能受用一部分
case-keep-replace　　替换文本的时候可以保留原来的命名风格
copy-path　　可以灵活地复制文件路径
change-case　　改变命名规则
sort-lines　　排序、删除重复行
set-syntax　　一种简单的命令方式来设置当前文件语法，与 Sublime 类似。
sublime-style-column-selection　　alt +单击跨行选择文本块，每行都有插入符号。
toggle-quotes　　快速切换字符串的单引号和双引号。
color-picker　　颜色选择器。
es6-javascript
js-hyperclick & hyperclick　　点击跳到变量或者 import 定义，js-hyperclick 依赖于 hyperclick。
pigments　　代码上直接显示颜色
linter-eslint　　 js静态检查
linter-csslint　　css静态检查
minimap　　代码预览图
platformio-ide-terminal　　集成终端,使用太顺畅了
indent-guide-improved　　 显示父元素首尾连线
atom-bracket-highlight　　括号选中的时候对应的亮一下
auto-id-class　　#+tab 直接出id=””,.+tab 直接出class=””,懒人必备
